# VR 射击建造综合游戏

# 

# 项目简介

基于 Unity 独立开发的 **VR 第一人称视角综合游戏**，无 AI 辅助、无现成教程照搬，全程自主拆解需求、设计架构、编码实现。核心融合「射击战斗、自由建造、物理滑翔、怪物交互」四大玩法，完整覆盖游戏业务闭环与工程化开发思维。

## 核心功能



### 战斗系统



* VR 枪械机制：开火、瞄准、换弹、子弹计数、音效动画联动
* 近战剑攻击：挥剑判定、粒子特效、怪物伤害碰撞检测
* 怪物 AI：血量管理、受击反馈、死亡动画、自动寻路（基础版）

### 建造系统



* 多类型建筑：支持墙壁、地板、楼梯、门的自由放置
* 智能吸附：网格对齐、90° 旋转、相邻建筑自动拼接
* 交互控制：建造确认、物体删除、层级管理

### 滑翔系统



* 物理驱动：左右转向、匀速下降、空气阻力模拟
* 视觉反馈：拖尾特效、相机震动、姿态同步

### 交互与 UI



* VR 射线交互：物体高亮、拾取判定、道具交换
* 状态提示：子弹数量、腰间备弹、操作反馈文本
* 菜单控制：建造菜单唤醒、武器切换、状态切换

## 技术栈



* 引擎：Unity 3D
* 开发语言：C#（3000+ 行纯手写源码）
* 核心技术：

  * 架构设计：单例模式、枚举状态管理、模块化分层
  * 交互逻辑：射线检测、碰撞回调、事件分发
  * 动画与物理：Animator 状态机、Rigidbody 物理引擎、插值平滑

## 操作说明



|按键|功能描述|
|-|-|
|M 键|打开 / 关闭建造菜单|
|G 键|拾取道具 / 交换弹夹|
|T 键|切换武器（枪 ↔ 剑）|
|Y 键|确认建造 / 建筑拼接|
|U 键|删除当前选中建造物体|
|Tab 键|切换滑翔状态|
|鼠标左键|枪械开火|
|鼠标右键|枪械瞄准|



## 项目特色



1. **纯自研开发**：无 AI 辅助、无教程照搬，从需求拆解到落地全程独立完成
2. **工程化思维**：严格区分业务逻辑与视图层，封装公共管理器（动画、事件、场景）
3. **解耦设计**：采用单例、枚举、事件分发，降低模块依赖，提升扩展性
4. **完整闭环**：覆盖「交互输入 → 业务处理 → 视觉反馈 → 状态存储」全流程
