## 项目介绍
本人独立用 Unity 做的一款VR综合小游戏，包含射击打怪、自由建造、物理滑翔三个核心模块，全程自己拆解需求、写逻辑、调参数，独立自主完成系统架构，工程量在4000行左右C#代码，没有套用完整成品模板。

核心技术特征：
- **模块化架构**：武器系统、建造系统、滑翔系统、敌人 AI、场景管理等为独立模块，通过全局管理脚本进行解耦。
- **单例模式管理**：基于单例模式实现武器管理、场景调度、事件分发等全局控制器。
- **复杂交互处理**：设计枪械弹夹与腰间子弹的状态转移逻辑，解决多对象状态同步问题；建造模块支持网格吸附、旋转放置。

## 主要功能

### 射击战斗
- 枪械瞄准、开火、换弹、子弹数量统计
- 近战剑攻击伤害判定、粒子特效
- 怪物受击、血量、死亡动画逻辑

### 自由建造
- 可放置地板、墙壁、楼梯、门等建筑
- 支持网格吸附、90度旋转
- 可以选中确认放置和删除已有建筑

### 滑翔功能
- 模拟重力和空气阻力，支持左右转向滑翔
- 搭配拖尾特效和相机震动，手感更真实
- 调整旋转速度与俯冲下降

## 技术栈
- 引擎：Unity 3D
- 语言：C#
- 用到：射线检测、单例管理、事件分发、状态枚举、模块化分层开发
- 架构 模块化分层开发，全局管理器解耦

## 操作按键说明
| 按键 | 功能 |
|------|------|
| M | 打开/关闭建造菜单 |
| G | 拾取道具/换弹 |
| T | 枪剑武器切换 |
| Y | 确认建造摆放 |
| U | 删除选中建筑 |
| Tab | 开启/关闭滑翔模式 |
| 鼠标左键 | 开枪攻击 |
| 鼠标右键 | 瞄准 |

## 运行说明
下载源码用 Unity 2021.3 及以上版本即可打开运行，支持电脑键鼠(方便在线调试)，也可适配VR手柄设备。
